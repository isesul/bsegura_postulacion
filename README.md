# bsegura_postulacion

## Instalacion

```sh
 git clone https://github.com/isesul/bsegura_postulacion.git
 cd bsegura_postulacion/
 docker-compose build
 docker-compose up
```

Abrir en el navegador la direccion http://localhost:3050
